5. Entire react application should be responsive
6. Host the application in any free hosting environments like netifly etc 

## Getting Started

### Prerequisites

- Node v14.x.x

### Build/ Run

1. `npm i`
2. `npm run build`
3. `npm run start`
4. navigate to `http://localhost:<port>` (defaults to 3000)
